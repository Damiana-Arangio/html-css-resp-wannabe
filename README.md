<h1 align="center">Responsive Wannabe</h1>

<p align="center">
  Esercizio svolto durante il corso Boolean per esercitarmi con le media queries CSS e l’adattamento di un layout a diverse risoluzioni.
</p>

## Obiettivo

- Partire dal layout esistente del repo `html-responsive-wannabe`.
- Rendere la pagina navigabile e leggibile alle seguenti larghezze:
  - **tablet**: fino a 768px;
  - **mobile**: fino a 480px.
- Non modificare l’HTML e il CSS già presenti, ma **aggiungere** solo nuove regole per il comportamento responsive.

## Media queries

- Aggiunta di media queries per:
  - adattare il layout alla visualizzazione tablet (≤ 768px);
  - adattare il layout alla visualizzazione mobile (≤ 480px);
  - migliorare il comportamento tra 768px e 1160px per evitare la scrollbar orizzontale (bonus).
